Advanced Usage

Overview

This vignette covers advanced features of lwdid: multiple estimators, sensitivity analysis, diagnostics, wild cluster bootstrap, randomization inference, data preprocessing, parallel computation, and result export.

library(lwdid)
data(smoking)
data(castle)

Multiple Estimators

lwdid supports four estimators, all applied to the transformed cross-sectional data:

Estimator estimator = Controls Required Double Robust
Regression Adjustment "ra" Optional No
Inverse Probability Weighting "ipw" Yes No
Doubly Robust "ipwra" Yes Yes
Propensity Score Matching "psm" Yes No

IPW

result_ipw <- lwdid(
  data = castle, y = "lhomicide", ivar = "sid", tvar = "year",
  gvar = "gvar", rolling = "demean", estimator = "ipw",
  controls = c("police", "income", "poverty"),
  aggregate = "cohort"
)

Transformation Comparison

res_dm <- lwdid(
  data = smoking, y = "lcigsale", ivar = "state", tvar = "year",
  d = "d", post = "post", rolling = "demean"
)

res_dt <- lwdid(
  data = smoking, y = "lcigsale", ivar = "state", tvar = "year",
  d = "d", post = "post", rolling = "detrend"
)

cat("Demean ATT:", round(res_dm$att, 4), "\n")
#> Demean ATT: -0.4222
cat("Detrend ATT:", round(res_dt$att, 4), "\n")
#> Detrend ATT: -0.227

For quarterly data, use "demeanq" or "detrendq" with the season_var parameter.

Variance-Covariance Estimation

vce Description
NULL OLS (homoskedastic)
"robust" / "hc1" HC1 heteroskedasticity-robust
"hc0" – "hc4" HC-family (HC3 recommended for small \(N\))
"cluster" Cluster-robust (requires cluster_var)

Data Preprocessing

aggregate_to_panel() collapses micro-level data to panel format:

agg_result <- aggregate_to_panel(
  data = survey_data,
  unit_var = "state",
  time_var = "year",
  outcome_var = "income",
  weight_var = "survey_weight",
  treatment_var = "treated",
  min_cell_size = 30
)
panel <- as.data.frame(agg_result)

Wild Cluster Bootstrap

For inference with few clusters, use vce = "bootstrap" with a cluster_var:

result_wcb <- lwdid(
  data = smoking, y = "lcigsale", ivar = "state", tvar = "year",
  d = "d", post = "post", rolling = "demean",
  vce = "bootstrap", cluster_var = "state",
  wcb_reps = 999
)
summary(result_wcb)

When vce = "cluster" is specified and the number of clusters is below 20, lwdid automatically upgrades to the Wild Cluster Bootstrap (auto_wcb = TRUE by default).

Randomization Inference

Exact p-values without distributional assumptions:

result_ri <- lwdid(
  data = smoking, y = "lcigsale", ivar = "state", tvar = "year",
  d = "d", post = "post", ri = TRUE, rireps = 500
)
cat("RI p-value:", result_ri$ri_pvalue, "\n")

Parallel Computation

Enable parallel processing for staggered designs:

result_par <- lwdid(
  data = castle, y = "lhomicide", ivar = "sid", tvar = "year",
  gvar = "gvar", rolling = "demean", aggregate = "cohort",
  parallel = TRUE, n_cores = 4
)

Or configure the parallel backend explicitly:

setup_parallel(n_workers = 4, strategy = "multisession")
# All subsequent lwdid() calls use parallel backend
result <- lwdid(...)
# Reset when done
setup_parallel(strategy = "sequential")
